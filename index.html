Write what is inside Bob's mind

In "Bob the simplebot", players can write real-life logic rules to guide Bob reach its destination and experience more and more difficult puzzles. By brainstorming through each one of these puzzles, you will be able to learn real-life logic concepts, symbolism and arguments.

The project

The project behind this game is the master thesis of an Imperial College student doing an MEng degree in Computing. The idea was to create a simple game to help teachers to teach students the basics of Logic. As they thinking through each puzzle, They will write and read simple rules that are then processed using a LPS framework. If you want to read more about LPS please read one of the original papers published by Robert Kowalski and Fariba Sadri: Towards a Logic-based Production System Language (2010), Reactive Computing as Model Generation (2015) and Programming in Logic Without Logic Programming (2016). This LPS framework provides a lightweight logic-based tool inspired by artificial intelligence and suitable for practical applications. JLPS - the Java version of LPS, developed by another Imperial master student (Alexandre Camus), is used as the underlying logic engine. Aside from logic, the very early game implementation is also developed by another Imperial MSc student (Matthieu Raymond) which is used as the foundation of my project.

Multiple game modes to explore different logic concepts and features!

Download the game

All platform

If you have the Java Runtime Environment (JRE minimum version: 1.7.0) installed on your machine (which you probably have), download this jar and double click on it or type in a terminal:
java -jar Bob_the_simplebot.jar

Windows

If you use Windows, you can download this executable file and just double click on it (it is portable). You might have a security warning as I do not own any valid signing certificate.

Mac

If you have Mac OS, you can download this application file and just double click on it (it is portable), however you might have to disable Gatekeeper. (The .app file is in the zip)
